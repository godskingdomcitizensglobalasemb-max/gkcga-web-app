## 🚀 Overview

GKCGA Platform is a full-stack React application designed to manage organization activities including user registration, event booking, mentorship programs, virtual summits, and community engagement. The platform supports multiple user roles (admin, mentor, user) with role-based access control.

## ✨ Features

### Public Pages
- Home page with organization overview
- About us and mission statement
- Program pillars and spheres
- Benefits and mentorship programs
- Projects and events showcase
- Resources library
- Contact form
- Virtual summits
- Volunteer registration
- Live streaming
- Community forum
- Ticket purchasing

### Authentication System
- User registration and login
- Password reset functionality
- Social media login (Facebook, Twitter, Google)
- Role-based access (Admin, Mentor, User)
- Protected routes

### User Dashboard
- Profile management
- Project submissions
- Mentorship requests
- Event registrations
- Payment history
- Notification center

### Admin Panel
- User management
- Mentor approvals
- Project moderation
- Event creation and management
- Analytics dashboard
- Content management system
- Payment monitoring

### Payment Integration
- Stripe payment gateway
- PayPal integration
- Paystack (African markets)
- Flutterwave support
- Ticket purchasing system

## 🛠 Tech Stack

### Frontend
- React 18
- React Router DOM v6
- Context API for state management
- CSS3 with responsive design

### Backend Services
- Firebase (Authentication, Database, Storage)
- Cloudinary (Media management)
- SendGrid (Email service)

### Payment Gateways
- Stripe
- PayPal
- Paystack
- Flutterwave

### Analytics & Monitoring
- Google Analytics
- Facebook Pixel
- Sentry (Error tracking)
